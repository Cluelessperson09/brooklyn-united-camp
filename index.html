📧 Email Notifications

Get an email when a note is added, a camper is transferred, or a camper is dropped. Uses EmailJS (free, no backend needed).

Setup (one-time, ~3 minutes):

  1. Go to emailjs.com → create a free account
  2. Add Email Service (connect Gmail) → copy the Service ID
  3. Create Email Template with variables: {{subject}}, {{message}}, {{to_email}} → copy the Template ID
  4. Go to Account → copy your Public Key
  5. Fill in the fields above and click Save
